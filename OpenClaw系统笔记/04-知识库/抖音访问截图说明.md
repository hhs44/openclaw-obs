## 📸 截图说明

这是使用 Playwright 浏览器自动化访问抖音首页的截图。

**技术栈：**
- Playwright 1.58.0
- Chromium 浏览器
- Node.js v25.6.1

**测试代码：**
```javascript
const browser = await chromium.launch({ headless: false });
const page = await browser.newPage();
await page.goto('https://www.douyin.com/');
await page.screenshot({ path: '/tmp/douyin-home.png' });
```
