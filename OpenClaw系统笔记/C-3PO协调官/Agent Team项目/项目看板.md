## 📝 使用流程

### 快速启动
1. **评估任务** - 判断是否需要 Agent Team
2. **设计团队** - 确定队友数量和角色
3. **创建执行** - 启动团队并监控
4. **结果整合** - 汇总输出并报告

### 决策树
```
任务请求
    ↓
任务是否可分解？
    ├─ 否 → 单 agent 执行
    └─ 是 → 并行能否提升？
              ├─ 否 → 单 agent 执行
              └─ 是 → 创建 Agent Team
```
