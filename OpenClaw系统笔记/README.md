## 🎯 使用指南

### 如何记录

**手动记录**：
```bash
./agent-obsidian-record.sh <agent> <内容> <类型>
# 示例：
./agent-obsidian-record.sh c3po "完成工作" work
./agent-obsidian-record.sh cortex "学习新知识" learn
```

**自动记录**：每天 21:00-23:50 自动运行

### 如何查看

1. **本索引**：主目录导航
2. **Agent 索引**：点击 Agent 名称查看专属索引
3. **直接浏览**：进入具体目录查看笔记

### 如何搜索

在 Obsidian 中使用搜索功能：
- 按 Agent：`tag:#c3po`
- 按类型：`tag:#work`
- 按日期：`file:2026-02-26`
